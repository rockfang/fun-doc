# 重构菜单项为子项目

项目基于 qiankunopen in new window

拆解后的目录结构如下，main是主项目，其他是子项目。目前一个一级菜单对应一个子项目

| -- main
| -- sub-dashboard
| -- sub-datacollect
| -- etc..

# 示例：创建dashboard子项目

以拆解dashboard项目为例，看下拆解和组装流程。

# 1. 创建子项目

目前主项目和子项目我们都用 vueopen in new window 技术栈 。以下是使用 vue-cli3open in new window创建项目流程。

vue create sub-dashboard
1

我们创建包含babel router vuex eslint的项目。

# 2. 子项目配置

  • 改造子项目路由 router/index.js

    把懒加载的路由变成直接加载，导出routes。代码如下：

//  router/index.js
import Vue from "vue";
import VueRouter from "vue-router";
import Analysis from "../views/Dashboard/Analysis.vue";
import IframeAnalysis from "../views/Dashboard/IframeAnalysis.vue";

Vue.use(VueRouter);

const routes = [
  {
    path: "/",
    redirect: '/analysis1'
  },
  {
    path: "/analysis1",
    name: "Analysis",
    component: Analysis
  },
  {
    path: "/analysis2",
    name: "IframeAnalysis",
    component: IframeAnalysis
  },
];

export default routes;

在我们的中台项目中，由于菜单栏都放在main中管理，子项目本身并没有导航按钮。因此子项目其实只需平铺配置好每个路由即可。

  • 配置main.js

暴露子项目的生命周期钩子，配置子项目渲染路由

import Vue from "vue";
import App from "./App.vue";
import routes from "./router";
import VueRouter from "vue-router"
import store from "./store";

Vue.config.productionTip = false;

// eslint-disable-next-line no-undef
__webpack_public_path__ = window.__INJECTED_PUBLIC_PATH_BY_QIANKUN__;

let router = null;
let instance = null;

function render(props = {}) {
  const { container } = props;
  router = new VueRouter({
    base: window.__POWERED_BY_QIANKUN__ ? '/sub-dashboard' : '/',//设置base
    mode: 'history',
    routes,
  });

  instance = new Vue({
    router,
    store,
    render: h => h(App),
  }).$mount(container ? container.querySelector('#app') : '#app');
}

if (!window.__POWERED_BY_QIANKUN__) {
  render();
}
export async function bootstrap() {
  console.log('[sub-dashboard] vue app bootstraped');
}

export async function mount(props) {
  console.log('[sub-dashboard] props from main framework', props);
  // storeTest(props);
  //mount方法中触发应用的渲染方法
  render(props);
}

export async function unmount() {
  //卸载应用
  instance.$destroy();
  instance.$el.innerHTML = '';
  instance = null;
  router = null;
}

// 单独开发环境
window.__POWERED_BY_QIANKUN__ || mount();

{
    name: 'sub-dashboard',
    entry: dev ? '//localhost:7003' : 'http://172.16.2.123:30203',
    container: '#subapp-container',
    activeRule: genActiveRule('/sub-dashboard'),
},
