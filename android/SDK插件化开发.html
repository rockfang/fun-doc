# 加载流程

壳代码提供sdk初始化方法，初始化功能:

加载sdk核心代码 执行sdk的初始化方法

核心代码加载流程:

解密核心代码 解密update下的核心代码 比对版本选择要加载的路径 dexclassloader加载核心代码

以上描述也说明了，如何构建插件化的sdk

  1. 构建核心代码
  2. 构建壳代码
  3. 构建资源目录
  4. 加密核心代码，整合壳代码及资源，进行打包发布。
