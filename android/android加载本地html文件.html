# 图片转base64再加载的方式:

即在android端获取图片的路径，并把图片转成base64编码，js端通过调用android端代码，获取base64编码并加载。

# 1. android端代码

图片转base64工具方法：

/**
 * 将图片转换成Base64编码的字符串
 */
public  String imageToBase64(String path) {
    if (TextUtils.isEmpty(path)) {
        return null;
    }
    InputStream is = null;
    byte[] data = null;
    String result = null;
    try {
        is = new FileInputStream(path);
        //创建一个字符流大小的数组。
        data = new byte[is.available()];
        //写入数组
        is.read(data);
        //用默认的编码格式进行编码
        result = Base64.encodeToString(data, Base64.DEFAULT);
    } catch (Exception e) {
        e.printStackTrace();
    } finally {
        if (null != is) {
            try {
                is.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }

    }
    return result;
}

获取图片路径：

String imgPath = Environment.getExternalStorageDirectory() + "/tencent/QQfile_recv/xuersi_banner.png";

定义js端调用android端的对象接口：可参考open in new window

private Object getHtmlObject(){
    Object insertObj = new Object(){

        @JavascriptInterface
        public String getImgBase64() {
            return imageToBase64(imgPath);
        }
    };

    return insertObj;
}
