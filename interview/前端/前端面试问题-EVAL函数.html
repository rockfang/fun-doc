# 前端面试问题-EVAL函数

eval函数就是一个代码解析器，把字符串解析成代码运行。

示例1

eval(’let a = 3; alert(a);’);
1

将会弹框显示数字3。

eval使得js高度灵活，如，可以让用户输入代码，页面执行。使得页面代码动态化

  <body>
    <textarea id=’ta’></textarea>
    <div id=”btn”>执行</div>
  </body>
  <script>
    let btn = document.getElementById(’btn’);
    btn.onclick = ()=> {
      let code = document.getElementById(’ta’).value;
      eval(code);
    }
  </script>

# EVAL在REQUIREJS中应用

在requireJs（AMD模块化规范的代表）中，就是利用强大的eval实现。 如异步加载三个模块：

define([”jquery”,”moduleA”,”moduleB”],function($,a,b){
})
1
2

eval(’任意内容’);

其原理是:

  • 通过指定type为文本方式引入js文件，<script src=”xxx.js” type=”text/html”></script>
  • 再通过eval(text)执行内容，实现异步加载的效果

# EVAL在定时器中的应用

可以把定时器中的函数写成字符串代码：

setTimeout(’alert(555)’,300)//会像正常函数一样执行alert(555)
1

代码正常执行，定时器内部就是通过eval(text)方式处理的

# 严格模式下EVAL的作用域

默认情况：

let a = 3;
function test() {
  eval(’var a = 20; alert(a)’);
 //等价于var a = 20
  console.log(a); //打印20。 因为eval中重新定义了a为20
}
test();

加上严格模式：use strict

’use strict’
let a = 3;
function test() {
  eval(’var a = 20; alert(a)’);
  console.log(a); //打印3。 因为严格模式下，eval中定义的a只在eval作用域中有效
}
test();
