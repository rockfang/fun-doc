# lodash使用

安装lodash

  "dependencies": {
    "lodash": "^4.17.20"
  }

使用报错

尽管在chrome中我们能看到，生效了，但由于我们使用的lodash是JavaScript版，IDE会提示无法找到模块

解决方式，找寻"类型翻译包"

@types/lodashopen in new window 就是lodash的一个types包，在github上查看open in new window，其实就是提供了申明lodashjs包中的类型的无数.d.ts文件；

关于TS中申明文件使用open in new window

Tips

使用第三方JavaScript库时，可同时搜索 Types librayName 找到它的types库。安装即可正常使用

# class-transformer

使用class-transformeropen in new window可以把json数据直接转成我们定义的类对象。

安装以来：

npm install class-transformer --save
npm install reflect-metadata --save

2

全局引入基础以来

import 'reflect-metadata';

使用plainToClass方法进行json数据转换

定义Model:

  name: string
  sqlStatement: string
  constructor(name: string, sqlStatement: string) {
    this.name = name
    this.sqlStatement = sqlStatement
  }

# class-validator

class-validatoropen in new window提供装饰器的方式实现数据的校验功能。

tsconfig.json开启experimentalDecorators功能

"experimentalDecorators": true,        /* Enables experimental support for ES7 decorators.*/
1

安装

npm install class-validator --save
1

使用：

//LabelInfo.ts
import { MaxLength, IsNotEmpty, MinLength } from 'class-validator'
class LabelInfo {
  @MinLength(2, {
    message: '标签名长度不能小于2',
  })
  @MaxLength(10, {
    message: '标签名长度不能大于10',
  })
  name: string

  @IsNotEmpty()
  sqlStatement: string

  constructor(name: string, sqlStatement: string) {
    this.name = name
    this.sqlStatement = sqlStatement
  }
  showMsg() {
    console.log(`name: ${this.name}  sqlStatement: ${this.sqlStatement}`)
  }
}
export { LabelInfo }
